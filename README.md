
# MiniShop 🛍️

A modern e-commerce frontend application built with React, TypeScript, Redux Toolkit, and Vite.

## Overview

MiniShop is a responsive shopping application that allows users to browse products, search and filter items, manage a shopping cart, and enjoy a smooth user experience with animations and modern UI practices.

This project was built to practice modern React development patterns, state management, form validation, routing, and TypeScript.

---

## Features

* Product listing from API
* Product search functionality
* Category filtering
* Shopping cart management
* Add products to cart
* Increase and decrease product quantity
* Remove products from cart
* Responsive design
* Smooth UI animations with Framer Motion
* Global state management using Redux Toolkit
* Form handling with React Hook Form
* Schema validation with Zod
* Client-side routing with React Router
* Reusable component architecture
* Type-safe development with TypeScript
